# Espanso Setup

This repository includes a set of reusable text expansion shortcuts using Espanso.

## Purpose
To speed up prompt writing by injecting reusable preprompts instantly.

## Installation
1. Install Espanso
2. Copy `espanso/base.yml` to your local Espanso config folder

## Available Shortcuts

### ;clean
Applies punctuation rules to avoid em dashes and ensure natural flow.

### ;pro
Enforces a professional and concise tone.

### ;clear
Improves clarity and avoids unnecessary complexity.

## Usage Example

;clean ;pro ;clear
Explain Docker for a beginner.
