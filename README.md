# AI Writing Workflow

A structured system for improving AI-generated text through reusable preprompts and writing standards.

## Purpose
This repository defines a minimal set of rules and templates to ensure consistent, natural, and professional writing output.

## Core Principle
Avoid artificial or mechanical text transformations by enforcing clear punctuation and natural sentence flow.

## Key Rule Example
> Do not use em dashes (—). Replace them with commas or periods where appropriate, and adjust sentence structure so the text flows naturally.

## Structure
- `docs/` → writing rules and standards
- `prompts/preprompts/` → reusable constraints
- `prompts/templates/` → task-based prompts
- `examples/` → before/after demonstrations

## Usage
Combine:
1. A preprompt (rule)
2. A template (task)
3. Your input
